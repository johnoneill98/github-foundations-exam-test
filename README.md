## ğŸš€ How to Use These Templates

### Creating an Issue
1. Go to the **Issues** tab
2. Click **New Issue**
3. Choose from the available templates:
   - Bug Report
   - Feature Request
   - General Issue
4. Fill out the form fields
5. Submit your issue

### Creating a Pull Request
1. Create a branch and make your changes
2. Push your branch to GitHub
3. Open a Pull Request
4. The PR template will automatically populate
5. Fill in the sections with relevant information
6. Submit your PR for review

### Using Discussions for Q&A
1. Go to the **Discussions** tab (if enabled)
2. Click **New Discussion**
3. Select the **Q&A** category
4. Use the template to structure your question
5. Submit and engage with the community
